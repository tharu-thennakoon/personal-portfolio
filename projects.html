My Projects

Finance Tracker

A personal finance management tool created in Python with a user-friendly GUI using Tkinter. This application enables users to track income, expenses, and budgets in real time, offering categorized transaction summaries and goal-setting features for effective financial planning.

Green Edge Website

The Green Edge website is a visually appealing and interactive platform developed using HTML5, CSS, and JavaScript. It showcases a commitment to sustainability and eco-friendly practices, featuring a modern design and responsive layout that adapts seamlessly to various devices. The website includes sections on green initiatives, educational resources, and community engagement, all enhanced with dynamic elements and animations. With a focus on user experience, this project effectively communicates the mission of promoting environmental awareness and encourages visitors to take part in sustainable practices.

Point of Sale System

A comprehensive Point of Sale (POS) system developed using Spring Boot for the backend and React for the frontend. This application facilitates seamless sales transactions, inventory management, and customer engagement. The backend handles data processing and business logic, ensuring secure and efficient transaction handling, while the React frontend provides an intuitive user interface for sales staff. Features include product scanning, payment processing, sales reporting, and real-time inventory updates, making it an essential tool for retail environments.
